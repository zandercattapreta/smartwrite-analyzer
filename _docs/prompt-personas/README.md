# Prompt Personas (Canonical)

This folder contains canonical, normalized persona definitions used by SmartWriter Analyzer.

Included personas:

- `helena-vasconcelos` — Senior Beta Reader & Manuscript Analyst
- `character-analyst` — Character-focused critiques
- `common-reader` — Clarity and engagement from an average reader perspective
- `critical-editor` — Structural and high-impact issues
- `devils-advocate` — Rigorous counterpoints and assumption testing
- `market-strategist` — Genre & market positioning
- `technical-reviewer` — Technical accuracy and consistency
- `genre-critic` — Genre conventions and fit

Notes:
- Legacy persona files are kept in `_docs/propmt personas/` (typo) — these normalized files are the canonical source for prompts and will be referenced by the prompt-versioning system.
- Each persona document includes: metadata (ID/icon), short description, system prompt, activation prompt and limitations.

_If you want to add or revise a persona, update the canonical file here and create a PR referencing the `v0.2.0` milestone._
